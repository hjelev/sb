Controls

Keyboard-first, all the way down

A taste of the keymap — press h in-app for the full help overlay.

Navigation

  • Enter / →open entry / preview file
  • ← / Bkspparent / leave mount
  • Spacemark / unmark entry
  • *toggle all marks
  • `toggle view modes
  • ~jump to home
  • .toggle hidden files
  • /filter folder
  • 0–9jump to bookmark

File ops

  • ccopy to clipboard
  • vpaste
  • mmove (cut + paste)
  • ddelete (confirm)
  • r / F2rename / bulk rename
  • nnew file or folder
  • xtoggle executable bit
  • page protect / unprotect

Power tools

  • GGit commit workflow
  • Ctrl+GAI commit message (in commit prompt)
  • fsearch overlay
  • gcontent search (rg)
  • Zarchive create / extract
  • SSSH / rclone remotes
  • Iintegrations panel
  • ;run a shell command
  • ^zdrop to shell here
